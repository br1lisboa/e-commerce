## DEV
1- Clonar el repositorio.

2- Crear una copia del archivo 
```.env.template```
y renombrarlo a 
```.env```
y cambiar las variables de entorno.

3- Levantar la base de datos
```docker compose up -d```

4- Instalar dependencias 
```npm install```

5- Correr el proyecto
```npm run dev```
